# Arkham_Helper_Bot
Arkham Helper/詭鎮奇談小幫手，能幫忙玩家利用LINE輕便的紀錄遊戲中角色及劇情資訊，以利遊戲進行。  
紀錄冒險日誌所使用的時區為：Asia/Taipei
目前所使用技術皆是試用版/免費版方案，可能出現流量限制或每月流量使用完畢的情況，盡請見諒。
